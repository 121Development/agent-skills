# Section Checklist

## Open Source (`templates/oss.md`)

- [ ] **Name & Description** - Clear, compelling
- [ ] **Installation** - Step by step, tested commands
- [ ] **Usage** - Common use cases with examples
- [ ] **API/Configuration** - Reference if applicable
- [ ] **Contributing** - How to help, code of conduct
- [ ] **License** - Which one, where to find full text
- [ ] **Changelog** - What's new in this version
- [ ] **Support** - Where to ask questions

## Personal (`templates/personal.md`)

- [ ] **What it does** - The problem it solves
- [ ] **Tech stack** - What you used and why
- [ ] **How to run** - If you want others to try it
- [ ] **Learnings** - What you discovered
- [ ] **Status** - Done, WIP, abandoned?
- [ ] **Screenshots/Demo** - Visual proof of life

## Internal (`templates/internal.md`)

- [ ] **Setup** - Get dev environment running
- [ ] **Architecture** - How it fits together
- [ ] **Runbooks** - Common operations
- [ ] **Deployment** - How it gets to production
- [ ] **On-call** - Who to ping, what to check
- [ ] **Decision records** - Why choices were made

## Config (`templates/xdg-config.md`)

- [ ] **What's here** - File inventory
- [ ] **Why** - Rationale for each file
- [ ] **How to extend** - Adding new configs
- [ ] **Gotchas** - Things that will bite you
- [ ] **Dependencies** - What else needs installing
